    • O que é o Github?

    O Github é uma plataforma de hospedagem de códigos-fontes e controle de versão compartilhado onde os usuários ao redor do mundo podem contribuir para formação e atualização de um projeto.

    Visualiza os arquivos que foram alterados.

    git status

    Prepara um arquivo para ser versionado

    git add nome_de_um_arquivo

    Gera a versão das modificações

    git commit -m "uma mensagem legal do que foi feito nos arquivos nesta versão"

    Envia para o github

    git push origin main
